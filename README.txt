●構成

「stable」フォルダーと「stable-audio」フォルダーに、最終的に完成したファイルが入っています。
「stable」フォルダーは「音なし」、「stable-audio」フォルダーは「音あり」の完成ファイル一式が入っています。
また「chapter-～」フォルダーは、各章に対応しています。
